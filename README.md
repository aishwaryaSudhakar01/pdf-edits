# PDF Edits

**One PDF. Every edit. One download.**

A browser-based PDF editor that batches all your changes into a single pass. Reorder, annotate, redact, sign, convert, and export — without your files ever leaving your device.

🔗 **Live app:** [pdf-edits.lovable.app](https://pdf-edits.lovable.app/)

---

## Why this exists

Most online PDF tools make you do one thing at a time: upload, crop, download, re-upload, add page numbers, download again. This editor loads your file once and lets you stack every edit, then exports a single clean PDF at the end. Everything runs locally in the browser, so nothing gets uploaded to a server.

## Features

**Edit**
- **Organize** — merge files and reorder pages
- **Split** — extract page ranges into separate files
- **Rotate** — rotate individual or all pages

**Enhance**
- **Compress** — reduce file size
- **Page Numbers** — add numbering
- **Watermark** — overlay text
- **Metadata** — edit title, author, and other PDF info

**Modify**
- **Black-out** — place a visual cover over content (note: this hides, it does not delete the underlying text)
- **Crop** — trim page margins

**Annotate**
- **Text** — add text anywhere
- **Highlight** — mark areas
- **Image Stamp** — place images on a page
- **Signature** — sign pages

**Convert**
- **Resize Pages** — A4, Letter, or custom dimensions
- **Image → PDF** — turn images into a PDF
- **PDF → Image** — export pages as images

## Privacy

All processing happens client-side in your browser. Your PDFs are never sent to or stored on any server.

> ⚠️ The **Black-out** tool is a visual cover only. It does not remove the underlying text from the file, so it should not be used to permanently redact sensitive information.
